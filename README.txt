La carpeta del juego contiene contiene los recursos de Imagenes y Sonidos utilizados
junto con las funciones necesarias para el funcionamiento:

FUNCIONES IMPLEMENTADAS EN EL JUEGO --------------------------------------------------------------
- CalibracionJuego.m		Funcion para calibrar la sensibilidad del juego antes de empezar
- Nivel1.m			Funcion que grafica el nivel 1 del juego
- Nivel2.m			Funcion que grafica el nivel 2 del juego
- Nivel3.m			Funcion que grafica el nivel 3 del juego
- Ganaste.m			Funcion que grafica la imagen indicando que ganaste
- GameOver.m			Funcion que grafica la imagen indicando que perdiste 

%---------------------------------------------------------------------------------------------------
Para EJECUTAR el juego se requiere correr el script principal llamado:
-Pixeles.m		main del juego
